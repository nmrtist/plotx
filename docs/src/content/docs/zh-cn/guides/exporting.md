## 导出图形

通过工具栏的导出菜单导出——范围为当前页、全部页或页码区间。

| 格式 | 用途 |
| --- | --- |
| SVG | 矢量图，可在插图软件中继续编辑 |
| PDF | 矢量图，适合投稿 |
| PNG / TIFF / JPEG | 位图，DPI 可在 72–1200 间调节（默认 300） |

预设覆盖常见期刊图幅——例如 *Single column · 89 mm · 600 dpi ·
TIFF*——并且导出前的合规检查会按所选预设标出字号与线宽不达标之处，让
问题在画板上就被修正，而不是被期刊编辑发现。

## 复制图形

*Copy figure*（复制图形，`Ctrl` + `C`，也可在导出菜单、命令面板或图框
右键菜单中触发）把选中的图框——未选中时为活动画布——直接复制到剪贴板，
无需先导出。在 Windows 上会同时发布位图（PNG + DIB）与矢量
（SVG + EMF）格式，粘贴目标应用自动选择最合适的格式：聊天软件粘贴为
位图，Word、PowerPoint、WPS 则粘贴为可编辑的矢量图。

## 导出数值数据

选中数据集后，可从 File 菜单、Data Ribbon 选项卡或命令面板选择
**导出数据…**。对话框只显示当前数据集实际存在的内容，并提供
**保存 CSV…**、**保存 TSV…**、**保存 XLSX…**和**复制 TSV**。

处理后的 NMR 数据可选择 Real、Imaginary 或 Magnitude 强度。真实 2D 与
伪 2D 均支持 **Matrix** 和 **Long**：Matrix 首行是 F2/ppm，首列是
F1/ppm 或伪 2D 的系列轴；Long 每行一个观测值，真实 2D 使用
`f1_ppm,f2_ppm,intensity`，伪 2D 则使用带名称和单位的系列轴、`ppm` 与
`intensity`。大型导出在后台生成。

从数据表导出的 CSV 或 TSV 会附带一个配套的 `.plotx-schema.json` 文件，XLSX 导出
则把同样的信息保存在隐藏工作表中。可见的列在 Excel、Origin 或 Prism 中正常打开，
而配套信息让 PlotX 日后能连同列类型、单位和误差棒一起重新打开该表。导出的 XLSX
文件只包含确定值，没有需要重新计算的公式。
