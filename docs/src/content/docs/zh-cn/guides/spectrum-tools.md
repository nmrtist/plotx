---
title: 谱运算与谱图对齐
description: 组合谱图，或对齐整个谱图系列。
---

有两个工具作用于多条谱而非单条谱：**谱运算**用两条已有的谱生成一条新
谱，**谱图对齐**平移整个系列使参考峰对齐。二者都可从 1D 谱的分析面板或
命令面板（<kbd>Ctrl</kbd>+<kbd>K</kbd>）打开。

## 谱运算

**谱运算**把 1D 谱组合成一个新的独立数据集——源数据集不会被修改。

提供四种运算：

- **A + k·B** 与 **A − k·B**——加上或减去第二个谱，并乘以系数
  *k*（默认 1）。微调 *k* 即可在扣除中压平溶剂峰，或平衡差谱。
- **A × k**——谱乘以常数。
- **A + c**——加一个常数偏移。

两个操作数必须是同一核种；核种不同时会拒绝并在对话框中显示原因。
若两谱的 ppm 轴不一致（点数或范围不同），第二个谱会线性插值到第一
个谱的轴上，重叠区间之外按零处理——对话框会提示这一情况。

结果是一条独立的频域谱，放置在自己的页面上，并以表达式命名（例如
`A − 0.9·B`）。它与其他 1D 谱行为一致——峰、积分、区域与导出均可
使用——且创建过程是单个可撤销步骤。

## 谱图对齐

**谱图对齐**把一系列 1D 谱平移，使共同的参考峰落在同一化学位移位置——
适用于反应监测、滴定等参考峰在多次采集间漂移的系列实验。

对话框作用于数据列表中的多选（少于两个被选中时作用于全部数据集）：

- 选择一个 **ppm 区间**；默认取当前活动谱的可见范围。每条谱在区间内
  最高的显著峰即其参考峰。
- 选择**对齐目标**：参考谱自身的峰位置（活动数据集为参考谱），或手动
  输入一个 ppm 值。
- 预览表逐一列出数据集、找到的峰位和将要应用的平移量。区间内没有显著
  峰、核种不同或非 1D 类型的数据集会被跳过，并显示原因。

平移通过每条谱的**参考**步骤写入——与单谱化学位移定标使用同一机制——
因此之后仍可在管线中查看和编辑。整个对齐操作是跨所有数据集的单个可撤销
步骤。
