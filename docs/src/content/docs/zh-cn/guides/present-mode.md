---
title: 演示模式
description: 全屏逐页展示你的页面。
---

演示模式把 PlotX 变成幻灯片：所有编辑面板消失，一个页面充满屏幕，居中
显示并加黑边——适合组会和快速讨论，无需导出幻灯片。

## 开始演示

在工具栏或命令面板（<kbd>Ctrl</kbd>+<kbd>K</kbd>）中选择 **Present
Full Screen**。演示从当前所在的页面开始。若尚未打开任何内容，PlotX 会
提示没有可演示的页面。

**Presentation Dark** 画布主题与演示模式很搭——它会把坐标轴文字放大到
投影可读的尺寸。演示前先应用它。

## 翻页

| 输入 | 动作 |
| --- | --- |
| `→` / `↓` / `Space` / `PageDown`，或单击鼠标 | 下一页 |
| `←` / `↑` / `PageUp` | 上一页 |
| `Home` / `End` | 第一页 / 最后一页 |
| `Esc` | 退出演示模式 |

翻到首页或末页后停止，不会循环。页面多于一页时，右下角显示
`当前页 / 总页数` 计数。

退出演示模式后，窗口和所有面板恢复原状。
