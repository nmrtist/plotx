表格在 PlotX 中是一等数据集：承载导入的表格数据、区域工具提取的数值，
以及拟合结果。

## 编辑

双击画板上的表格即可打开可编辑的工作表。工作表是电子表格式的网格：带行号、
列宽可拖拽，表头显示列名并在下方标注单位和类型。空单元格显示为弱化的短横线，
`NaN` 与无穷值也以弱化样式显示。网格只绘制当前可见的行窗口，超大表格依然流畅。

单击选中单元格，用方向键移动；双击、按 **Enter** 或 **F2**、或直接输入即可开始
编辑。**Enter** 提交并下移，**Tab** 提交并右移，**Esc** 取消。输入不合法时编辑框
保持打开并以红色描边提示，悬停可查看原因。点击行号选中整行；**Ctrl+C** 复制当前
单元格或整行。状态栏显示表格规模与当前选中位置。

每次编辑都可撤销；撤销和重做会在表格的编辑历史中前后移动，而不是每次都替换整张
表。文本、布尔、日期和空单元格的编辑方式与数字完全相同。

## 转换与刷新表格

右键单击列头，可围绕该列重塑表格。菜单提供 **Rename（重命名）**、
**Sort ascending（升序）** 和 **Sort descending（降序）**、**Keep only this
column（只保留该列）**、**Filter out empty rows（过滤空值行）**、**Duplicate as
computed column（复制为计算列）** 和 **Count rows per value（按值计数）**；浮点列
还会增加 **Mark non-finite as missing（把非有限值标为缺失）**，以及当该列带有可
换算单位时的 **Convert unit（换算单位）** 子菜单。表格含多列同一类型时，可用
**Unpivot matching columns** 把它们堆叠成名称/取值行，文本列可用 **Pivot using
these names** 把其取值变成新列名。工作表工具栏中的 **Combine** 菜单可与项目中的
另一张表执行合并或联接。每次转换都会生成一张新表，原表保持不变。

转换在后台运行：工作表会显示耗时和 **Cancel（取消）** 按钮，取消或失败的转换不会
替换原表。由另一张表生成的表会获得 **Refresh（刷新）** 操作——它会重跑生成该表的
步骤，再把你之后的单元格编辑重新应用到结果上。如果来源数据发生变化、导致被编辑
的行无法再匹配，或某列的类型或单位不再吻合，刷新会停止并报告冲突，而不会猜测哪
一行对应哪一行。

这些转换也可以作为批处理任务无人值守地运行——参见
[自动化](/zh-cn/guides/automation/)和[命令行](/zh-cn/reference/cli/)。

## 图表类型

选中由表格生成的图，在对象检查器的**图表类型**画廊中选择。Ribbon 的 **Figure**
页签中的 **Chart** 可直接跳转：它会选中表格对应的图并打开检查器。

选图表就是选它回答的问题：沿轴跟踪数值用 **Line**；跨类别比较数量用
**Bar** 或 **Grouped bars**；看单列数值的分布用 **Histogram**；一眼比较
多列的分布用 **Box** 或 **Violin**；展示整体中的占比用 **Pie**；一次看
整个数值矩阵用 **Heatmap** 或 **Surface 3D**。

- **Line（散点/折线）**——每列一组散点，按 x 轴排布；已有拟合时叠加拟合曲线。
- **Bar（柱状图）**——所选单列在数值 x 轴上绘制为填充柱。
- **Grouped bars（分组柱）**——每行一组、组内各列并排；勾选 **Stacked**
  改为堆叠（正值向上、负值向下堆叠）。
- **Histogram（直方图）**——所选单列的数值分布。默认按
  Freedman–Diaconis 规则自动分箱；取消勾选 **Auto bins** 可指定箱数。
- **Box（箱线图）**——每列一箱：中位数、四分位箱体、1.5 × IQR 须线，
  以及逐个绘制的离群点。
- **Violin（小提琴图）**——每列一个核密度轮廓，内嵌四分位条
  （每列至少需要两个不同数值；退化列回退为原始散点）。
- **Heatmap（热图）**——整张表作为色图矩阵，行自上而下排列；
  在检查器中选择色图。
- **Pie（饼图）**——所选单列的每一行为一个扇区，并标注百分比。
  负值与缺失行不参与。
- **Surface 3D（3D 曲面）**——把表格视为高度场，以固定正交视角绘制；
  可在检查器中调整方位角/仰角与色图。

分组柱、箱线、小提琴、热图与 3D 曲面一次使用全部列；柱状图、直方图和
饼图读取画廊旁选中的单列。所有图表在屏幕显示与 SVG/PDF/EMF 导出中
完全一致。

## 由表格作图

误差棒来自一列——它保存另一列的不确定度。PlotX 分析工具生成的表格已经带有这些
关系，导入的表格则会带上其文件所描述的关系——对称 ±σ、分开的上下误差，或带置信
水平的置信区间。要给自己的数据添加误差棒，可在 CSV 或 TSV 中，紧跟某个 y 列放一
个名为 `<y>_sigma` 的列，保存每个点的不确定度（取零或正值）；导入时 PlotX 会把它
作为对称标准差关联到该 y 列。绘图前，PlotX 会检查数值列与其不确定度的单位是否
兼容。散点图、柱状图和未堆叠分组柱会把这些绘制为纵向误差棒，并扩展 y 轴以保证
误差棒完整可见。

需要用数学模型拟合其中的 x-y 数值时，在**分析**选项卡的**曲线拟合**组中选择**拟合曲线**。画布右上角会打开曲线拟合
任务卡片：选择模型，再选择拟合全部曲线或一条选定曲线，然后选择**运行拟合**。
有关内置模型、自定义模型、共享（全局）参数、权重与稳健损失选项以及拟合诊断，
请参阅[曲线拟合](/zh-cn/guides/curve-fitting/)。拟合曲线及其结果会保留在表格中。
使用**导出数据…**并选择**曲线拟合参数**，可导出参数、诊断、协方差以及
逐点预测和残差。
每次完成拟合都会记录它实际使用了哪些行、排除了哪些行以及排除原因，因此后续编辑
表格不会悄悄改变该结果所依据的观测点。曲线拟合导出会列出这些纳入和排除的行。

**导出数据…**也可以把完整表格导出为 CSV、TSV、XLSX 或剪贴板 TSV。空单元格保持为
空；`NaN`、`+Inf` 和 `-Inf` 会写成这些字面标记，因为它们是真实取值而非空白。以上
每种格式都会把表格的列类型、单位和误差棒关系与数据一起携带——CSV/TSV 用配套的
`.plotx-schema.json`，XLSX 用隐藏工作表，复制 TSV 用剪贴板内容——因此只有当你手工
构造没有这些信息的文件时，才需要 `<y>_sigma` 这样的列名。

**谱峰拟合**是另一项独立功能：当绘制出的曲线包含相互重叠、需要分解的
谱峰时使用它——参见[分析工具一览](/zh-cn/guides/choosing-a-tool/)。

要对表格的列比较分组、检验相关性、检查正态性或运行方差分析，请在
**Analyze（分析）** 标签中选择 **Statistics（统计）**。参见
[统计分析](/zh-cn/guides/statistics/)。
