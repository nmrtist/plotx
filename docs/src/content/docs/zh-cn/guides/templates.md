当一系列实验需要完全相同的处理时，把管线保存一次再重复应用——保存为可
分享的**配方文件**，或随 PlotX 设置存放的**命名模板**。

## 配方文件

管线（含群延迟设置）可从处理面板保存为可移植的 `.plotxproc` 配方文件，
并在本机或同事的机器上应用到其他数据集。配方也是
[自动化工作流](/zh-cn/guides/automation/)与[命令行](/zh-cn/reference/cli/)
中处理步骤的基本构件。

## 命名模板

常用配方可保存为**命名模板**，随应用设置一同存放，无需文件对话框即可
随手取用。

- 处理面板中的**另存为模板…**把当前数据集的管线按你输入的名称保存。
  名称已存在时对话框会给出提示，按钮变为*覆盖*。
- **应用模板…**列出已保存的模板。与当前数据集不匹配的模板（例如把
  双轴模板用于 1D 谱）会标记为不兼容并显示原因，无法应用。应用模板
  是单个可撤销的步骤。
- **应用到…**可把模板一次应用到多个数据集：打开的确认对话框会列出
  你多选的数据集（未多选时列出全部数据集）及各自的兼容性，并把模板
  作为**一个**可撤销步骤应用到其中兼容的数据集。
- **删除**移除模板；点击一次进入待确认状态，再次点击确认。

*Save processing template…* 与 *Apply processing template…* 也可以在
命令面板（<kbd>Ctrl</kbd>+<kbd>K</kbd>）中调用。
