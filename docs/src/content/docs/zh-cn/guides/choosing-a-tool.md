PlotX 有几个名字相近的分析工具。从你想测量什么出发：

| 你想…… | 使用 | 适用对象 | 产出 |
| --- | --- | --- | --- |
| 找出并列出谱中的峰 | **峰**（`P`） | 1D 谱 | 峰列表 |
| 跟踪某信号在有序系列中的强度变化 | **区域** | 伪 2D 与堆叠系列 | 联动系列表，每区域一列 |
| 测量 2D 谱中交叉峰的体积 | **积分**（`I`） | 真 2D 谱（COSY、HSQC 等） | 含归一化体积的积分表 |
| 把重叠的谱峰分解为组分 | **谱峰拟合**（`D`） | 任意 1D 谱线 | 每峰的位置、高度、宽度、面积 |
| 把拟合出的峰归组为多重峰并给出 J 耦合 | **多重峰分析**（谱峰拟合内） | 1D 谱 | 多重峰表与期刊格式描述 |
| 用数学模型拟合 x-y 数据 | **曲线拟合** | 数据表（含系列表） | 带不确定度与诊断的参数 |
| 组间比较、相关性或正态性检验 | **统计** | 数据表 | 已保存的统计结果 |

两个工具都像能用时的判断准则：

- **峰 vs 谱峰拟合**——峰工具标记位置；谱峰拟合把重叠谱线分解为
  Lorentz/Gauss/伪 Voigt 组分并给出面积与不确定度。先标峰；峰重叠或
  需要定量面积时再拟合。
- **区域 vs 积分**——区域在系列的*每个成员*上测量*同一个 1D 区间*；
  积分测量*单个真 2D 谱*中的*矩形*。DOSY 或弛豫数据是系列 → 用区域；
  HSQC 是真 2D 谱 → 用积分。
- **谱峰拟合 vs 曲线拟合**——谱峰拟合作用于谱线形；曲线拟合作用于表
  格化的 x-y 数值（衰减、滴定、IV 曲线）。伪 2D 分析两者都用：区域把
  衰减曲线提取到表中，曲线拟合再去拟合它们。
