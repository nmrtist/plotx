title: 2D 体积积分
description: 在真实二维 NMR 谱中测量矩形峰体积。
---

COSY、HSQC、HMBC、NOESY、ROESY 和 TOCSY 等真实二维等高线谱可以用矩形
范围进行体积积分。这些积分独立于伪 2D 序列使用的**区域**，也不会改变现有的
1D 积分流程。

## 创建和编辑积分

1. 打开真实二维谱，按 `I` 选择**积分**工具。
2. 拖出矩形框住交叉峰。任一方向不超过约三个屏幕像素的微小拖动会被忽略。
3. 拖动矩形内部可移动它；拖动边或角可调整大小。八个控制柄都会限制在谱图范围内。
4. 松开指针后计算体积。拖动时矩形会实时更新，但只在提交编辑时重新计算体积。

按 `Esc` 取消正在进行的编辑。按 `Delete` 或 `Backspace` 删除所选积分。右键菜单还
提供**设为参照**和**删除**。创建、移动、调整大小、重命名、更换参照和删除均可撤销及
重做。

积分表列出 F2/F1 边界、原始体积、归一化体积、计算时使用的显示模式和基线设置。
可以重命名条目、选择参照并设置参照的数值。使用
**导出数据…** → **积分表**可保存为 CSV/TSV，或复制为 TSV。

## 体积的含义

PlotX 对计算积分时屏幕上正在显示的曲面进行积分：

- **实部（Real）**保留相敏谱实部曲面的正负号，因此负峰叶会得到负体积。
- **幅值（Magnitude）**对非负的绝对值曲面积分，常用于 HMBC 等幅值模式谱。

每个积分都会保存当时选择的模式。以后更改显示不会重新解释已有数值；处理流程发生
变化（包括相位变化）时，则会按该积分保存的模式重新计算。

原始体积是在实际 F2 和 F1 ppm 轴上进行的二维梯形积分，单位为 intensity·ppm²。
它会计算矩形边界处的部分网格，并支持升序、降序和非均匀坐标轴。在均匀网格上，
该数值等于数字点求和再乘以 F2 与 F1 的网格宽度，因此同一谱内各积分之比与直接
数字求和之比相同。

## 基线校正

基线校正默认**关闭**，此时矩形直接对显示曲面积分。定量分析时，每个积分可单独启用
以下局部校正之一：

- **常数（Constant）**：减去矩形周边网格点的中位数。
- **平面（Plane）**：用周边点拟合倾斜平面，并在整个矩形中减去该平面。

所选基线方法会被保存并写入数据导出。基线校正会改变计算出的原始体积；归一化则
永远不会修改原始体积。

## 参照归一化

第一个积分自动成为参照。PlotX 对每个积分报告：

`归一化体积 = 原始体积 / 参照原始体积 × 参照值`

参照值默认为 `1`。需要时可把它设为定量权重，例如 HSQC 参照代表一个 CH₂ 基团时
设为 `2`。实部模式的有符号体积中，若某峰与参照符号相反，归一化值可以为负数。

若参照体积相对于其他积分实际上接近零，则无法可靠归一化。表格和图中标签会显示
`—`，而不是不稳定的比值。删除参照后，剩余的第一个积分会成为新参照。

## 限制与放置建议

- 仅支持矩形，不支持椭圆、多边形、等高线套索、自动峰范围和解卷积。
- 重叠矩形会各自积分完整面积，因此重叠部分会在两个数值中重复计算。
- 积分范围需手动放置；2D 积分没有自动峰检测。
- 若矩形包含 t1 噪声条纹、COSY 对角线或其他伪影，它们也会被照常积分。应尽量避开。
- 积分不会分离同一矩形中的重叠峰。

对于由一维曲线堆叠而成的序列，请改用[伪 2D 分析](/zh-cn/guides/pseudo-2d/)
及其**区域**流程。
