伪 2D 数据集是在某个参数变化下采集的一叠 1D 谱。PlotX 导入时从采集参数
中读取这个变化参数：梯度强度系列标记为 DOSY，延迟系列标记为弛豫
（T1 还是 T2 由你选择的拟合模型决定）。

## 工作流

1. 导入伪 2D 数据集。
2. 在**分析**选项卡的**区域**组中选择**绘制区域**，然后在感兴趣的峰上画出区域。
3. 在区域任务卡片中选择**继续到系列表**。
4. 系列表打开时会同时出现**曲线拟合**任务卡片。确认建议的模型，选择拟合全部
   曲线或一条选定曲线，然后选择**运行拟合**。

## 拟合模型

- 单指数衰减
- 反转恢复（T1），`a + b·exp(−x/T)`
- 饱和恢复
- Stejskal–Tanner 扩散衰减（DOSY）
- 双指数与拉伸指数
- 线性

DOSY 还可以使用正则化反拉普拉斯变换（ILT），可生成完整的
化学位移 × 扩散系数图。图在后台构建，期间可继续操作；按**取消**可丢弃
该构建。同一数据集在当前构建完成或取消前不能重复构建同类图。
构建期间修改该数据集的处理参数会取消构建——否则图与谱图将不再对应。
状态栏会在发生取消时提示；处理修改完成后重新构建即可。

## 关于强度的说明

提取的强度是带符号的（定相后的实部投影），而不是模值，因此反转恢复
数据无需折叠即可正确拟合。
