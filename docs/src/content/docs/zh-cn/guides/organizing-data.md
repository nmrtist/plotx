## 数据浏览器

把主侧栏切换到 **Data**，即可查看项目中全部内容的可搜索树。导入的数据
集是根节点；切片、投影、谱运算结果、区域表、谱峰拟合表和多重峰表显示在
来源数据集的 **Derived data** 下。展开 **Analysis** 可查看单个峰、积分、
区域、谱峰拟合、多重峰和已拟合的表列。

链接图标表示结果有多个来源——例如由两条谱共同生成的表。每个来源下的
条目都是同一个数据集，并非副本：选择、高亮、重命名和打开数据表会保持
同步。悬停图标可查看全部来源，也可在右键菜单中使用 **Reveal sources**。

搜索不区分大小写，并保留结果的完整祖先路径；清空搜索后会恢复此前的展开
状态。

## 选择与打开

单击数据集或引用可聚焦。按住 `Shift`、`Ctrl` 或 `Cmd` 单击可扩展多
选——多选正是在一个图中堆叠多条谱、或把处理模板一次应用到多个数据集
的方式。双击数据集打开其数据表。单击分析项会聚焦其所属数据集，双击则
跳转到相应图和分析工具。

用很旧版本的 PlotX 保存的项目可能把部分派生结果显示为顶层数据集；它们
仍完全可用。

## 画板视野

在 **Canvas** 模式下，侧栏下方是已命名的 **Board views**——保存下来的
画板视野，可一键返回。浏览数据时会隐藏。

## 保存项目

`Ctrl` + `S` 打开项目保存选项。`.plotx` 项目文件保存整个会话——导入的
数据、处理、分析结果和排版——重新打开时恢复原状，也可以作为单个文件
分享。文件内容与版本处理见[文件格式](/zh-cn/reference/file-formats/)。

**Preferences → General → Project backup copies** 会在项目旁以隐藏文件
保留指定数量的完整历史保存，误覆盖后可以恢复。
