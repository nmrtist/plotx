---
title: 自定义拟合模型
description: 用 .plotxfit 模型语言编写、编辑和分享自己的拟合模型。
---

当[内置模型](/zh-cn/guides/curve-fitting/#内置模型)与你的实验不匹配时，
可以自己编写。自定义模型的行为与内置模型完全一致：出现在模型列表中，
使用同样的求解器和权重，结果以同样的方式导出。

## 创建与管理

在任意模型旁选择 **Clone** 可创建一份可编辑副本，随后以 `.plotxfit`
定义进行编辑并 **Save to model library**。自定义模型会与内置模型并列出
现在模型列表中；用 **Edit custom model** 保存新修订，用 **Delete**
删除。内置模型在克隆前为只读。

自定义模型保存在你的 PlotX 配置目录中，可作为普通文件分享——模型文件
是纯数据，绝不运行代码。

## 模型语言

编辑器会在你输入时即时校验定义并报告未分类的符号；只有有效模型才能保存。模型有三种形式。

显式模型为每个响应赋一个表达式：

```text
let rate = 1 / T
y1 = a1 * exp(-rate*x)
y2 = a2 * exp(-rate*x)
```

隐式模型为每个响应给出一个代数等式：

```text
y^2 = a*x + b
```

ODE 系统使用导数方程，初始条件单独声明：

```text
d(A)/d(t) = -k1*A
d(B)/d(t) = k1*A - k2*B
```

标识符是区分大小写的 ASCII 名称；显示名和单位可以使用 Unicode。表达式支持
`+ - * / ^ **`、比较、`&&`、`||`、`!`、`if(condition, yes, no)`、三角/反三角/双曲函数、
`exp`、`ln`、`log10`、`sqrt`、`abs`、`erf`、`gamma`、`min`、`max`，以及常数 `pi` 和 `e`。
裸 `log` 会被拒绝，因为它的底数在不同领域含义不一——请写 `ln` 或 `log10`。

## 参数与常量

自由参数可设上下界；固定参数不进入优化；受约束参数是其他参数的表达式，循环依赖会被拒绝。
初值可以是数字，也可以是由数据计算的规则：`min`、`max`、`mean`、`median`、`quantile`、
`span`、起始/中段/末段斜率，以及插值 `x_at_y`。

模型还可以声明逐数据集常量；没有默认值的常量属于必需输入。ODE 系统可用自动、自适应非刚性
或 BDF 刚性方法积分，并接受初始时刻两侧的观测时间。
