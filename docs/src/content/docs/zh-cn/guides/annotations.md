title: 标注
description: 画板上的文本、面板序号、形状、箭头与分组。
---

图形很少不带标签或箭头就投稿。PlotX 的标注对象与图形一起放在画板上，
一起移动、一起导出，并且完全可撤销。

## 文本与面板序号

- **文本**（`T`）——自由文本标签或说明。在对象检查器中设置字号、颜色、
  对齐与加粗。
- **面板序号**——同一种对象，但默认样式为图形分图序号调校：加粗、
  8 pt，适合期刊版面。可从 Ribbon 和命令面板使用。

编辑标签时，`Enter` 提交，`Shift` + `Enter` 换行，`Esc` 取消。

## 形状

- **矩形**（`R`）与**椭圆**（`O`）——圈出关注的特征；均可设置描边颜色、
  描边宽度和可选的填充色。
- **直线**（`L`）与**箭头**——在拖动的两个端点之间画一条直线连接；
  箭头在末端加箭头头部。

在画布上拖动即可创建形状，之后可像任何图框一样移动或缩放。

## 样式默认值

调好一个对象的样式后，可以把它设为同类新对象的默认样式（"set as
default for new …"），这样一批标签无需逐个调整就能保持一致。这些默认值
在当前会话内有效；坐标轴与标题文字尺寸是文档级设置——见
[Figure Typography](/zh-cn/guides/layout-and-export/#作图样式与排印)。

## 选择、分组与锁定

- `Ctrl` + `A` 选中页面上的全部对象；`Esc` 逐级退出选择。
- `Ctrl` + `G` 把所选对象编组，之后一起选中、一起移动；
  `Ctrl` + `Shift` + `G` 取消编组。
- 对象可在右键菜单中锁定以防误编辑，也可以隐藏。
- `Delete` 或 `Backspace` 删除所选标注对象。

标注的对齐、分布与层叠顺序与图框完全一致——见
[画板](/zh-cn/guides/layout-and-export/)。
