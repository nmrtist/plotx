本页介绍 PlotX 窗口的主要区域，以及从原始数据到成品图形的典型流程。

## 窗口一览

- **主侧栏**（左）——数据集与项目结构。
- **画布**（中）——承载图形的无限画板，按网格对齐的页面组织。
- **副侧栏**（右）——针对当前所选数据的上下文工具：处理、峰、区域等。

在 **View** 菜单或 **View** Ribbon 中可隐藏任意一侧侧栏，为画布腾出空间。

## 菜单与任务 Ribbon

Windows 和 Linux 的标题栏把应用图标、**File**、**Edit**、**View**、**Insert**、
**Help** 菜单和窗口控制按钮合并为一行；拖动空白处可移动窗口，双击可最大化。
macOS 使用系统全局菜单栏，并包含符合平台习惯的 PlotX 应用菜单和 Window 菜单。

**File** 菜单中的 **Open Recent** 子菜单列出最近打开或保存过的文件、文件夹和
项目；尚未加载数据时，欢迎页也会显示同一份最近列表。**Help** 菜单中的
**User Manual** 会在浏览器中打开本手册。

菜单下方的 Ribbon 是高频命令入口，不是完整菜单的重复。选择 **Data**、**Process**、
**Analyze**、**Figure**、**Arrange** 或 **View** 可查看对应工作阶段的分组命令；使用 **Collapse ribbon** 可折叠为
仅显示任务页签。窗口变窄时，低优先级分组会整体移入 **More**；达到最小宽度时命令区会
自动折叠，而不会缩小文字或点击目标。Ribbon 下方的上下文摘要显示当前画布、对象、数据集、
任务和工具。

点击 **Search commands** 可打开命令面板。菜单、Ribbon、快捷键和命令面板共享相同的
可用与选中状态。

**Process** 和 **Analyze** 页签还会把副侧栏切换到对应的工具组，但不会重新打开你
已隐藏的侧栏。

## 数据浏览器

主侧栏有两种模式：**Canvas** 列出图形与页面，**Data** 显示每个数据集及
由它派生的结果。单击数据集可聚焦，双击打开其数据表。数据树、多选与已保
存画板视野的完整介绍见[组织数据](/zh-cn/guides/organizing-data/)。

## 典型流程

1. **导入**：把文件拖进窗口，使用 **File** 菜单，或在 Ribbon 的 **Data**
   任务页中选择导入命令。
2. **处理**：在处理面板中编辑管线——各步骤按顺序应用，结果实时预览。
3. **分析**：用峰与区域工具提取数值，结果出现在联动表格中。
4. **排版**：在画板上排布图形并**导出**。

[第一张图](/zh-cn/getting-started/first-figure/)用一个真实数据集走完这
四步。

## 导航

平移与缩放在任何工具下都可用：

- **滚轮**——缩放光标所在的图。
- **中键拖动**或 **Space + 拖动**——平移。
- **双击**——坐标轴自动适配范围。
