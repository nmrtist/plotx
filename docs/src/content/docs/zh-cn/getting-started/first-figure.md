本页用一个 1D NMR 数据集走完从导入到导出的全过程。每一步都可以推广：
同样的流程适用于 Bruker 文件夹、JCAMP-DX 文件、膜片钳记录或普通 CSV
表格。

## 1. 导入

把数据文件——例如 JEOL `.jdf`——拖到 PlotX 窗口上，或从 **File** 菜单
选择 **Open File…**。Bruker TopSpin 实验则使用 **Open Folder…** 并选中
实验目录。

数据集出现在主侧栏中，并自动放置到画板上，已在自己的页面里绘制完成。

## 2. 处理

打开 Ribbon 的 **Process** 页签。新的 1D 数据集已带有标准管线——切趾、
零填充、FFT、相位校正、基线校正——并默认启用自动相位，多数情况下屏幕
上的谱图已经可用。

需要调整时，在处理面板中编辑步骤：每次修改都实时预览且可撤销。两处
调整覆盖大部分场景：

- 基线起伏时，启用**基线校正**步骤。
- 自动相位不理想时，打开**相位校正**步骤，实时预览下调节 φ0 / φ1。

完整步骤列表见[数据处理](/zh-cn/guides/processing/)。

## 3. 分析

按 `P` 选择**峰**工具，拖动图上的阈值线：松开时高于阈值的峰会被检测出
来。自动检测不足的位置可手动添加、移动或删除峰标记。

提取出的数值——峰列表、积分、拟合参数——会进入与谱图联动的数据表。
本走查到峰拾取为止；[分析工具一览](/zh-cn/guides/choosing-a-tool/)介绍
其余工具及其适用场景。

## 4. 排版页面

点击页面左上角上方的尺寸标签（或 **Figure** Ribbon 页签的 **Canvas
Size & Settings…**），选择目标期刊的单栏预设。页面宽度会对齐到期刊的
制图规范；坐标轴文字已是印刷尺寸。

需要面板序号或标注时，按 `T` 选择文本工具并点击画布。见
[标注](/zh-cn/guides/annotations/)。

## 5. 导出

打开工具栏的导出菜单，选择矢量格式——SVG 便于继续编辑，PDF 适合投
稿——或栅格预设，例如 *Single column · 89 mm · 600 dpi · TIFF*。导出
预检会在写出前提示字号或线宽违反所选预设的问题。

要快速粘贴到文档里，`Ctrl` + `C` 会把所选图框同时以位图和矢量复制到
剪贴板。

## 6. 保存项目

`Ctrl` + `S` 把一切——数据、处理、分析和排版——保存为一个 `.plotx`
文件，之后可以重新打开或发给同事。见
[文件格式](/zh-cn/reference/file-formats/)。
