`Ctrl` 在 macOS 上对应 `Cmd`。文本输入框持有键盘焦点时，快捷键不生效。

## 工具

单键、无修饰键：

| 按键 | 工具 |
| --- | --- |
| `V` | 选择 |
| `Z` | 缩放（框选缩放） |
| `I` | 积分（根据数据集使用 1D 区间或真实 2D 矩形） |
| `P` | 峰 |
| `S` | 切片 |
| `D` | 谱峰拟合 |
| `T` | 文本 |
| `R` | 矩形 |
| `O` | 椭圆 |
| `L` | 直线 |

## 导航

平移与缩放是常驻层——任何工具下都可用，作用于光标所在的图；光标位于
空白处时作用于画板。

| 操作 | 效果 |
| --- | --- |
| 滚轮 / 双指捏合 | 缩放光标所在的图（双轴） |
| `Shift` + 滚轮 | 仅缩放 x 轴 |
| `Alt` + 滚轮 | 仅缩放 y 轴 |
| 在坐标轴带上滚动 | 仅缩放该轴 |
| `Ctrl` + 滚轮 / 捏合 | 改为缩放画板 |
| 中键拖动或 `Space` + 拖动 | 平移图（位于空白处或按住 `Ctrl` 时平移画板） |
| 在坐标轴带上拖动 | 框选该轴的范围并缩放至所选区间 |
| 双击图内 | 双轴恢复完整范围 |
| 双击坐标轴带 | 仅恢复该轴 |
| `F` | 缩放画板以适配所选图框（未选中时适配全部） |
| `Enter` | 缩放画板至所选页面或工作表 |

## 选择与编辑

| 操作 | 效果 |
| --- | --- |
| `Ctrl` + `S` | 打开项目保存选项 |
| `Ctrl` + `Z` | 撤销 |
| `Ctrl` + `Shift` + `Z` 或 `Ctrl` + `Y` | 重做 |
| `Ctrl` + `A` | 全选页面上的对象 |
| `Ctrl` + `G` | 编组所选对象 |
| `Ctrl` + `Shift` + `G` | 取消编组 |
| `Delete` 或 `Backspace` | 删除所选标注对象；在峰 / 积分工具中删除所选的峰或区域 |
| `F2` | 重命名所选数据集或画布 |
| `Esc` | 取消进行中的拖动；否则逐级回退选择，直至清空 |
| `Ctrl` + `C` | 把选中的图框（未选中时为活动画布）以位图 + 矢量格式复制到剪贴板 |
| `Ctrl` + `Shift` + `V` | 把剪贴板中的分隔文本（逗号、制表符或分号）粘贴为新数据表 |
| `Ctrl` + `,` | 打开首选项 |
| `Ctrl` + `K` 或 `Ctrl` + `Shift` + `P` | 打开[命令面板](/zh-cn/reference/command-palette/) |

编辑画板便签时：`Enter` 提交，`Shift` + `Enter` 换行，`Esc` 取消。

## UI 缩放

PlotX 会为每台显示器自动选择清晰可辨的 UI 缩放（首选项 → 外观中的
**UI scale** 设置）；以下快捷键以 10% 为步长微调，调整结果按显示器分别
记忆。

| 操作 | 效果 |
| --- | --- |
| `Ctrl` + `+` / `Ctrl` + `-` | 增大 / 减小当前显示器上的 UI 缩放 |
| `Ctrl` + `0` | 恢复自动缩放 |

## 演示模式

| 操作 | 效果 |
| --- | --- |
| `→` / `↓` / `Space` / `PageDown` | 下一页 |
| `←` / `↑` / `PageUp` | 上一页 |
| `Home` / `End` | 第一页 / 最后一页 |
| `Esc` | 退出演示模式 |
