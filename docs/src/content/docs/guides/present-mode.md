---
title: Present mode
description: Show your pages full screen, one at a time.
---

Present mode turns PlotX into a slideshow: every editing panel disappears and
one page fills the screen, letterboxed and centered — useful for group
meetings and quick reviews without exporting slides.

## Start presenting

Choose **Present Full Screen** from the toolbar or the command palette
(<kbd>Ctrl</kbd>+<kbd>K</kbd>). Presenting starts from the page you are
currently on. If nothing is open yet, PlotX tells you there is nothing to
present.

The **Presentation Dark** canvas theme pairs well with present mode — it
enlarges axis text to slide-legible sizes. Apply it before presenting.

## Navigate

| Input | Action |
| --- | --- |
| `→` / `↓` / `Space` / `PageDown`, or a mouse click | Next page |
| `←` / `↑` / `PageUp` | Previous page |
| `Home` / `End` | First / last page |
| `Esc` | Exit present mode |

Navigation stops at the first and last page rather than wrapping around. With
more than one page, a `current / total` counter appears in the lower-right
corner.

Leaving present mode restores the window and every panel exactly as you left
them.
